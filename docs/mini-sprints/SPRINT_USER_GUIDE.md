# Sprint User Guide

You do not need exact magic words for sprint tracking.

These are preferred examples, not required incantations.

## Common Requests

Create a sprint:
- `Create a sprint plan for this work`
- `Turn this into a sprint`

Track a sprint:
- `Track this sprint in the dashboard`
- `Link all prompts for this sprint`

Audit a sprint:
- `Audit this sprint against implementation`
- `What is actually done in this sprint`
- `What are the next best steps for this sprint`

Create prompts for a sprint:
- `Create the prompts needed for this sprint and link them back to the sprint`

Clarify before mutating:
- `If the sprint request is ambiguous, ask before updating the registry`
