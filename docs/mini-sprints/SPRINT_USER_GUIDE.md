These are preferred examples, not required incantations.

## Common Requests

Create a sprint:
- `Create a sprint plan for this work`
- `Turn this into a sprint`

Track a sprint:
- `Track this sprint in the dashboard`
- `Link all prompts for this sprint`

Audit a sprint:
- `Audit this sprint against implementation`
- `What is actually done in this sprint`
- `What are the next best steps for this sprint`

Create prompts for a sprint:
- `Create the prompts needed for this sprint and link them back to the sprint`
- `Create the next prompts for the next available sprint phase and use exact sprint linkage`
- `Open this sprint and give me the next prompt-generator request for planned work`
- `Open this sprint and give me the fix-prompt generator for current sprint problems`

Add your own sprint context from the dashboard:
- `Save this panel on the sprint's next-phase prompt panel`
- `Attach this extra context to the sprint audit prompt`
- `I want this feature request saved on the sprint so an agent can officially append it later`

Clarify before mutating:
- `If the sprint request is ambiguous, ask before updating the registry`

Normalize an older sprint:
- `Audit this sprint and normalize it to the current sprint workflow`
- `Fix this sprint so phases and tasks are normalized before generating more prompts`
