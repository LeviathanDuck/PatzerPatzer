# Patzer Pro — Big Picture Features down the line

- The website in the future should be to be loaded on mobile and use styling similar to lichess to accomplish this

- In the future, the users of this website should be able to import games and have the analysis saved locally

- The admin of the site should have a special log in area, where the user data can be saved to and “sync’d” that way if someone analyzes a bunch of games, they should be able to access that data on a different device

- We will start with a special, potentially secret admin login ability, keeping in mind in the future we may want to have users login themselves, and not start from a refreshed state every time they come back.
