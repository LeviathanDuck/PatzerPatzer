# CCP-679 — Opponents Left-Panel Label Truncation and Filter Default Fixes

## Summary

Fix truncated tool labels in the left-panel nav of the Opponents opening tree page, and change the default time filter to "All Time" in Prep Report and any other surfaces that use that filter.

## Context

The left-hand panel on the Opponents page contains tool nav labels that get cut off due to space constraints. The labels need to be shortened to fit cleanly. Additionally the Prep Report tool defaults to a time filter that is not "All Time" — that should be corrected.

## Tasks

| Current label   | Replace with |
|-----------------|--------------|
| Opening Tree    | Tree         |
| Repertoire      | Repertoire (fix truncation — shorten or wrap as needed) |
| Prep Report     | Report       |

Find where these labels are rendered in the Opponents/openings left-panel nav and apply the shorter strings.

### 2. Set default time filter to "All Time" in Prep Report

In the Prep Report tool (and any other surface on the Opponents page that uses a time-range filter), change the default selected value from whatever it currently is to "All Time".

## Scope

- Touch only the files that render the left-panel nav labels and the Prep Report default filter value
- Do not refactor surrounding code
- Maximum 1–3 files

## Validation
