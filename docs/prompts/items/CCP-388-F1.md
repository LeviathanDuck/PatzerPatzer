## Context

CCP-388 was linked to the tactical profile task but audit evidence shows no tactical breakdown view exists in `src/stats/view.ts`. The `GameSummary` type (`src/stats/types.ts`) includes `missedMomentCount` per game. Task 10 from the sprint spec requires surfacing a tactical profile panel on the stats page.

**Note on missed-moment type breakdown**: Check whether `GameSummary` stores missed-moment type breakdown (swing, collapse, missed mate separately). If it does not, and if adding those fields to the type and extraction logic would touch more than 3 files, implement the panel using only the aggregate `missedMomentCount` field and note the type breakdown as a follow-up item.

## What This Prompt Must Build

In `src/stats/view.ts`, add a tactical profile section per sprint Task 10:

**Content**:
- Missed moments per game (average across all analyzed games: `sum(missedMomentCount) / gameCount`)
- Total missed moments across all analyzed games
- If `GameSummary` has type breakdown fields (swing / collapse / missed mate counts): show per-type breakdown as a sub-list
- If trend data is sufficient (20+ games): show missed-moment rate trend using the same moving-average SVG approach as the trend charts (Task 8 / CCP-386-F1). If CCP-386-F1 has extracted a reusable chart helper, use it.

**Minimum sample**: require 10+ analyzed games before showing this section; hide entirely if below threshold — no empty placeholder.

**Rules**:
- Snabbdom `h()` only
- Touch only `src/stats/view.ts` (plus `src/stats/types.ts` if type breakdown fields already exist but are not used, or if a trivially small addition enables the breakdown)
- Do not add extraction logic if it requires modifying `src/stats/extract.ts` or `src/engine/batch.ts` — note that as a separate follow-up instead
- No chart library

## Validation

- Tactical profile section renders on stats page when 10+ games are analyzed
- Shows average missed moments per game
- Shows total missed moments
- Type breakdown shown if fields exist in `GameSummary`
- Missed-moment rate trend shown when 20+ games are available
- Section completely hidden (not even an empty container) when fewer than 10 games
