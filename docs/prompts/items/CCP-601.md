Fix: Also check whether the currently loaded study's ID matches the requested `id`.

```typescript

```

to:

```typescript
if (!detailLoaded() || studyDetail()?.id !== id) {
```

This ensures `loadStudyDetail(id, redraw)` is called whenever the requested study
differs from the one currently in memory.
