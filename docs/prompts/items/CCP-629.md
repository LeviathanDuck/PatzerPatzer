# CCP-629 — Mistake Detection: replace severity pills with gradient slider

## Context

The Mistake Detection modal currently uses three pill buttons (Inaccuracy, Mistake, Blunder)
to select the minimum severity threshold for Learn From Your Mistakes candidates.

Replace those pills with a 3-step slider that uses a green → amber → red gradient track,
tick labels below the track, and a marker indicating the Lichess default position.

## Target files

- `src/header/index.ts` — replace pill button group with slider vnode
- `src/styles/main.scss` — add gradient track, tick label, and default marker styles

## Changes required

### src/header/index.ts

Remove the `h('div.detection-modal__pills', ...)` block (the three pill buttons).

Replace it with a slider group:

```
h('div.detection-modal__severity-slider', [
  h('input.severity-range', {
    attrs: { type: 'range', min: 0, max: 2, step: 1,
             value: ['inaccuracy','mistake','blunder'].indexOf(cfg.minClassification) },
    on: {
      input: (e: Event) => {
        const idx = parseInt((e.target as HTMLInputElement).value, 10);
        const val = (['inaccuracy','mistake','blunder'] as const)[idx];
        setRetroConfig({ minClassification: val });
        redraw();
      },
    },
  }),
  // Lichess default marker at position 1 (Mistake = 50% along the 0–2 range)
  h('span.severity-range__default-mark', {
    attrs: { title: 'Lichess default: Mistake' },
  }),
  h('div.detection-modal__severity-ticks', [
    h('span', 'Inaccuracy'),
    h('span', 'Mistake'),
    h('span', 'Blunder'),
  ]),
])
```

The existing `detection-modal__value` span (showing `loss ≥ N%`) and the description
paragraph must remain unchanged — only the pill group is replaced.

### src/styles/main.scss

Add styles for the new slider and its supporting elements.

The slider track must use a CSS gradient from green (`#4caf50`) through amber (`#f0a500`)
to red (`#e53935`) to communicate increasing strictness.

The default marker is absolutely positioned at 50% (Mistake position in a 3-step slider)
and should be visually distinct — a small downward triangle or vertical tick above the track.

Tick labels sit below the track in a flex row with `justify-content: space-between`.

The `.detection-modal__severity-slider` wrapper must be `position: relative` to anchor
the default marker.

Cross-browser note: styling `input[type=range]` requires both `::-webkit-slider-runnable-track`
and `::-moz-range-track` rules. The thumb requires both `::-webkit-slider-thumb` and
`::-moz-range-thumb`.

### What NOT to change

- Do not change the `missedMateDistance` slider
- Do not change the Blown Wins, Defensive, or Punish families
- Do not change `detection-modal__value` span logic (already shows `loss ≥ N%`)
- Do not change the description paragraph text
