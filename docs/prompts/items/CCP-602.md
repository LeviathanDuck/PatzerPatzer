**Bug (Glyph quick-select mispositioned — main.scss line 9939):**
`.glyph-quick-select` uses `position: absolute; bottom: 48px; left: 50%` to float above
the board column. Its parent `.study-detail__board-col` has no `position: relative`,
so the popup positions relative to the nearest positioned ancestor (which may be the
viewport or an outer container), appearing in the wrong place on screen.

Fix: Add `position: relative` to `.study-detail__board-col` in `main.scss` (~line 9939):

Before:
```scss
&__board-col {
  display: flex;
  flex-direction: column;
  gap: 8px;
  flex-shrink: 0;
  width: min(45vh, 45vw);
}
```

After:
```scss
&__board-col {
  display: flex;
  flex-direction: column;
  gap: 8px;
  flex-shrink: 0;
  width: min(45vh, 45vw);
  position: relative; // anchor for .glyph-quick-select absolute positioning
}
```
