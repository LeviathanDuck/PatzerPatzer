1. Read the child prompt body at `docs/prompts/items/<ID>.md`
2. Run `npm run prompt:start -- <ID>`
3. Execute the work described in the prompt
4. Run `npm run prompt:complete -- <ID> --checklist "..."` (use the checklist from the prompt body's Lifecycle section)
5. Proceed to the next step only after the current child is fully complete
