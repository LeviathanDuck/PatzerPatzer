# Patzer Pro

Patzer Pro is a web app for analyzing personal chess games and generating targeted practice puzzles.

## Stack

Frontend
- React
- Tailwind
- react-chessboard
- chess.js

Backend
- Node.js
- Express
- SQLite

Engine
- Stockfish WASM
